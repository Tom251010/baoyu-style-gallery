如何指定： 在 Cursor 里直接说技能 + 参数，例如 「用 baoyu 小红书，风格 notion，布局 dense」或 「信息图 --style technical-schematic --layout funnel」。 不指定时会按内容自动推荐。出图默认 FoxCode（gpt-image）。

1. 小红书 · 视觉风格（baoyu-xhs-images）

指定：--style <name>　例：风格 notion

2. 小红书 · 布局（baoyu-xhs-images）

指定：--layout <name>　可与风格自由组合

3. 信息图 · 视觉风格（baoyu-infographic）

指定：--style <name>　默认 craft-handmade

4. 信息图 · 布局（baoyu-infographic）

指定：--layout <name>

5. 幻灯片风格（baoyu-slide-deck）

指定：--style <name>

6. 文章配图风格（baoyu-article-illustrator）

指定：--style <name>　可与 --type 组合

7. 封面风格预设（baoyu-cover-image）

指定：--style <preset>　或分别指定 --palette / --rendering / --type

8. 漫画布局（baoyu-comic）

指定：--layout <name>　画风另用 --art / --tone

指定方式速查

技能 Cursor 说法示例 关键参数 参数说明
小红书 用 baoyu 小红书，--style notion，--layout dense --style --layout --preset
  • --style风格：视觉美学，如 --style notion / --style cute / --style chalkboard
  • --layout布局：信息密度与结构，如 --layout dense / --layout list / --layout flow
  • --preset预设：风格+布局打包，如 --preset knowledge-card
信息图 做信息图，--style technical-schematic，--layout funnel --style --layout --aspect
  • --style风格：视觉美学，如 --style technical-schematic / --style craft-handmade
  • --layout布局：信息结构，如 --layout funnel / --layout iceberg / --layout bento-grid
  • --aspect画幅：比例，如 --aspect landscape / --aspect portrait / --aspect 3:4
封面 封面 --palette warm + --rendering flat-vector，或 --style blueprint --style --palette --rendering --type
  • --style预设：色板+渲染简称，如 --style blueprint
  • --palette色板：配色，如 --palette warm / --palette cool / --palette mono
  • --rendering渲染：画法，如 --rendering flat-vector / --rendering hand-drawn
  • --type类型：构图类型，如 --type hero / --type conceptual / --type typography
幻灯片 做 PPT，--style sketch-notes --style
  • --style风格：整套幻灯片视觉预设，如 --style sketch-notes / --style blueprint / --style corporate
文章配图 给文章配图，--type infographic，--style blueprint --type --style --preset
  • --type类型：信息结构，如 --type infographic / --type flowchart / --type comparison
  • --style风格：视觉美学，如 --style blueprint / --style notion / --style watercolor
  • --preset预设：类型+风格打包，如 --preset tech-explainer
漫画 知识漫画，--art manga，--tone warm，--layout cinematic --art --tone --layout --style
  • --art画风：线条与造型，如 --art manga / --art realistic / --art ink-brush
  • --tone基调：情绪氛围，如 --tone warm / --tone dramatic / --tone action
  • --layout布局：分镜结构，如 --layout cinematic
  • --style预设：画风+基调打包，如 --style ohmsha / --style wuxia / --style shoujo
